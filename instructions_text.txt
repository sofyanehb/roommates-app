3.  RÈGLES  
 
Fichier projet.sql — Contenu obligatoire 
 
1. Exporter avec phpMyAdmin : Structure + Données 
2. Inclure en tête : CREATE DATABASE IF NOT EXISTS et USE nom_bdd; 
3. Inclure au minimum 3 à 5 enregistrements de test dans chaque table 
4. Vérifier que le script SQL s'importe sans erreur avant de le remettre 
 
4.  CONTENU DU DOSSIER /doc 
 
 
4.1  Captures d'écran  (dossier captures/) 
 
Règles pour les captures : 
 
• Format : PNG ou JPG, résolution lisible (minimum 1024px de large) 
• Nommage : page_login.png, page_dashboard.png, page_ajout_produit.png… 
• Capturer toutes les pages de l'application sans exception 
• Inclure des captures avec données réelles (pas des pages vides) 
• Insérer les captures dans le rapport avec une légende explicative sous chacune 
 
4.2  Diagrammes  (dossier diagrammes/) 
 
Diagramme Nom de fichier attendu 
MCD — Modèle Conceptuel de Données mcd.png / mcd.pdf 
MLD — Modèle Logique de Données mld.png / mld.pdf
